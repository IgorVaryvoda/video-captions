About This Demo

This demo showcases videos and audio files that have been transcribed using the OpenAI Whisper API. Here's what makes this demo special:

Transcription Features
  • Accurate speech-to-text conversion
  • Support for multiple languages
  • Proper punctuation and capitalization
  • Timestamps for each spoken segment
  • Works with both videos and audio files
Player Features
  • Synchronized captions with video and audio
  • Interactive transcript with clickable segments
  • Spotify-style lyrics display for audio files
  • Toggle captions/lyrics on/off
  • Adjustable playback speed
  • Downloadable VTT caption files
